This repository contains all course related code projects undertaken over the entire semester of CSC591: Game Engine Design Foundations by Dr. David Roberts at NC State University Raleigh

1. The projects were individual assignments and focused on creating a game engine from scratch
2. It uses the Processing java library for rendering and Java7 for the underlying engine logic
3. The game engine is made up of -
  a. Event Management
  b. Customizable game object
  c. Multiplayer support through networked socket programming
  d. Replay system with fast-forward and slow-motion
